Topic: 
    High speed rail vs airplanes


Goal statement:
    This project aims to compare the time efficiency of high speed rail and airplanes over different distance.


highspeedrail_vs_airplane/
    src/
        generate_random_demand.py                   # Generate random travel demand
        generate_access_time_to_airport_station.py  # Randomly compute access time to airport or station
        generate_preboarding_time.py                # Randomly compute waiting and processing time
        generate_ride_time.py                       # Randomly compute in-vehicle travel time
    data/
        compute_total_time.py                       # Compute total travel time for each mode
        analyze_statistics.py                       # Statistical comparison between rail and air
        access_time_table.csv                       # Access times to airports or train stations
    results/
        distance_vs_time.png                        # distance vs time comparison
